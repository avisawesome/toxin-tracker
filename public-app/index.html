About Food Toxin Labels

This database provides information about toxins present in common foods. Just like nutrition labels help you make informed decisions about nutrients, toxin labels can help you understand potential exposure to various toxins in your diet.

Each toxin has a "Daily Value" (DV) which represents the maximum amount that is considered safe to consume in a day. The percentage shows how much of that daily value is present in one serving of the food.

Understanding the Label

  • Serving Size: All toxin values are based on this amount of food
  • Amount: The actual measurement of each toxin in the serving
  • % Daily Value: The percentage of the maximum safe daily intake

Note: This database is for educational purposes. For dietary or health concerns, please consult with healthcare professionals.
